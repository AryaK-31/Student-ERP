# Arya - School ERP management

## Getting started

In the root of the project folder run following commands

```bash
## For mac
nvm use
## For windows
nvm use 18
## Installation of dependencies
yarn install
## Run vite project locally
yarn nx run serve
```

Now go to [http://localhost:2401](http://localhost:2401)

## How to build

```bash
yarn nx run build
```

## How to preview

```bash
yarn nx run preview
```

## How to lint (eslint)

```bash
yarn nx run lint
```
